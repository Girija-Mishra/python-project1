📰 Fake News Headline Generator

A simple Python project that generates random, funny “breaking news” headlines using predefined subjects, actions, and places. This project demonstrates basic Python concepts such as lists, loops, string formatting, and the random module.

📌 Project Overview

The Fake News Headline Generator creates humorous headlines by randomly combining:

A subject (celebrity, public figure, or random character)

An action

A place or object

Each time the program runs, it generates a new headline and allows the user to continue generating more until they choose to stop.

🎯 Features

Random headline generation

Infinite loop until user exits

User interaction using input()

Clean and simple Python logic

Beginner-friendly project

🛠️ Technologies Used

Python

Random Module

📂 How It Works

Predefined lists of:

Subjects

Actions

Places/Things

The program:

Selects one item randomly from each list

Combines them into a formatted headline

Prints it to the console

Asks the user whether they want to generate another headline.

▶️ How to Run the Project

Make sure Python is installed.

Save the script as headline_generator.py

Run:

python headline_generator.py
